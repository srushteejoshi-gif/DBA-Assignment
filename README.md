# DBA-Assignment
This repository contains the dataset and colab files for Database and Analytics Assignment 

SQL in R and R Analytics: Investigates the raw data using SQL and charts to prove where NorthStar is losing money and losing customer trust.
Data Processing and Analysis: Cleans up all the messy errors and merges the separate files into one modern, organized dataset.
MongoDB Implementation: Builds a fast, secure, and future-proof database that lets management view a complete delivery history instantly.
The .csv files are the ones required to run the colab files attached.
•	orders.csv: Customer purchases, order values, and pickup/drop-off zones.
•	deliveries.csv: Logistics data with timestamps, fuel costs, and driver ratings.
•	app_events.csv: Technical logs of driver’s interactions and system latency.
•	complaints.csv: Customer feedback and financial compensation records.
•	drivers.csv: Personal details and vehicle types for the delivery fleet.
•	customers.csv: Profiles and contact information for the users.
•	warehouse.csv: Locations and capacities of the starting points.
•	items.csv: Catalog of products being delivered (weights, categories).
•	zones.csv: Mapping data for city boundaries.
•	routes.csv: Pre-calculated paths for delivery optimization.
